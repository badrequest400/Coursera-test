About

What is this site?

The main purpose of this site is to practice HTML, CSS and JavaScript skills learnt on from a course by Yaakov Chaikin on coursera.org. I also used Bootstrap 3, and I learned a lot from w3schools.com tutorials. It demonstrates my current knowledge, hence the name: DEMOSITE.

What can I do here?

Not much, actually, since it doesn't have any backend. It is a responsive, you can try setting the width of the browser or perhaps open it from a phone. It might rearrange items and resize them. For example the about, news, gallery and contact buttons on the top will be hidden in a dropdown menu if the width of the browser is lower than 768 pixels. This dropdown menu does not disappear when you click on an item, I know. Please click on the menu button to close it. Another thing I am actually proud of is the expanding search bar. In small display width (between 768 and 991 pixels), there wasn't enough room for the searchbar. So if you click inside it, it will expand and hower over the menu buttons. Pretty cool, huh?

You can see some news articles, I tried to look for interesting and meaningful topics. The content was shamelessly stolen. The code for the gallery was also shamelessly copied from w3schools' "How to" section. I does not understand how it works right now, but to be honest I didn't even try to. The contact form at the bottom doesn't work either. I hope one day it will all work function as it should and I will receive tons of fascinating messages from Nigerian princes and hot local girls! I also added a map, and I decided to use OpenStreetMap. Why OpenStreetMap? Because screw Google, that's why.
